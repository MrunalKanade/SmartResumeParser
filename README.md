\# Smart Resume Parser



A Python-based Resume Parser that extracts candidate information from PDF resumes.



\## Features



\- Email Extraction

\- Phone Extraction

\- Skills Detection

\- Education Detection

\- Experience Detection

\- Resume Scoring

\- CSV Export

\- JSON Export



\## Tech Stack



\- Python

\- Streamlit

\- PyMuPDF

\- Pandas

\- spaCy
